엑셀 데이터 한글문서 자동삽입 프로그램 ㅇ0ㅇ ver 0.0.2

한글파일에서 바꾸기 원하는 영역을 ctrl+k -> ctrl + e로 지정하세요. 필드이름을 워드의 칼럼값(맨 위 첫번째 줄의 로우 값들)이랑 매칭시키세요.
이 작업이 끝났으면 작업할 한글문서에서 아무런 값도 없는 빈페이지를 마지막 페이지에 하나 둔 후 저장하세요.
프로그램을 실행한 후 엑셀파일과 한글파일을 설정한 후 실행버튼을 누르면 됩니다.

혹시 프로그램이 튕긴다면 실행중인 한/글 프로그램을 모두 종료하고 이용해보세요.

진행중에 한/글에서 어떤 팝업창이 뜬다면 아무 허용버튼을 눌러주세요.

팝업창이 뜨는 것이 맘에 들지 않거나 제대로 작동되지 않는다면 한/글 오토메이션용 보안승인모듈을 등록해보세요. 좀 더 원활하게 이용하실 수 있습니다.
----------------------보안모듈등록방법-----------------------------
한/글 오토메이션용 보안승인모듈 다운로드한다(https://www.hancom.com/board/devdataView.do?board_seq=47&artcl_seq=4085&pageInfo.page=&search_text=)
dll 파일을 복사해 HNC폴더에 Automation_Module폴더(이름 자유)를 만들고 넣는다
dll 파일의 전체 경로 복사를 한다. (shift + f10 후 a(경로로 복사))
win키 + r => regedit실행
컴퓨터\HKEY_CURRENT_USER\SOFTWARE\Hnc\HwpAutomation\Modules에(경로에서 없는 폴더가 있다면 키 생성해서 만들면 됨) 새로만들기
 -> 문자열 값(한글에서 불러오는 모듈 이름이 됨)을 SecurityModule이라고 이름을 줌
오른쪽 마우스 클릭 -> 수정 -> 값데이터에 복사한 전체 경로를 넣어줌

잘 모르겠다면 https://www.youtube.com/watch?reload=9&v=2Cv16_ZO5rk을 참고해주세요.


-------------------------작업할 때 유용한 한글 단축키---------------------------

shift + alt + w: 자간 넓히기(스페이스바로 간격띄워야할때 대신 이용)
